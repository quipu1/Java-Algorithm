# 기수 정렬



## 핵심 이론

값을 비교하지 않는 특이한 정렬

값을 놓고 비교할 자릿수를 정한 다음 해당 자릿수만 비교

시간 복잡도: O(kn)

k는 데이터의 자릿수



10개의 큐를 이용함

각 큐는 값의 자릿수를 대표함

일의 자리를 기준으로 데이ㅓㅌ 저장

일의 자리르 기준으로 데이터 정렬

십의 자리르 ㄹ기준으로 데이터 저장

십의 자리를 기준으로 데이터 정렬

이런 방식으로 진행됨



큐에 집어넣는 방식, 3일 경우는 10의 자리를 0으로 처리하는 부분 등이 까다로움



코테에는 자주 나오지 않지만 어려운 문제에 가끔 나옴
