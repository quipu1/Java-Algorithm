# 구간 합



## 구간 합

합 배열을 이용하여 시간복잡도를 더 줄이기 위해 사용하는 특수한 목적의 알고리즘

코테 사용 빈도 높음



### 핵심 이론

구간합 알고리즘을 활용하려면 합 배열을 구해야 함



#### 합 배열

S[i] = A[0] + A[1] + ... + A[i]

합 배열은 기존 배열을 전처리한 배열



#### 합 배열 공식

S[i] = S[i-1] + A[i]



#### 구간 합 구하는 공식

i에서 j까지 구간 합 => S[j] - S[i-1]





처음 주어지는 배열의 값이 바뀌지 않는 이상 합배열이 바뀌지 않음

배열의 값이 바뀌는 경우는 다른 알고리즘을 사용!
